## Most used commands

### General

- `git init`
- `git clone <repo_url>`
- `git status`
- `git add -A`

### Changing branches

- `git checkout -` Switches to last branch
- `git checkout <branch>`

### Rebase

- `git pull --rebase <remote> <branch>`

### Set current branch to push default

- `git config push.default current`
